# HTML DOM

[HTML DOM学习传送门](https://www.w3school.com.cn/jsref/dom_obj_document.asp)

DOM 是一项 W3C (World Wide Web Consortium) 标准

HTML DOM 是 HTML 的标准*对象*模型和*编程接口*。它定义了：

- 作为*对象*的 HTML 元素
- 所有 HTML 元素的*属性*
- 访问所有 HTML 元素的*方法*
- 所有 HTML 元素的*事件*

**换言之：HTML DOM 是关于如何获取、更改、添加或删除 HTML 元素的标准**。
